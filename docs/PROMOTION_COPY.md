## 📣 版本四：技术控版

🔥 CCMage v1.2.0 技术亮点大公开

🏗️ **架构升级 - 多引擎 AI 系统**
• 后端新增 783 行代码，3 张数据库表
• 前端新增 391 行代码，完整类型定义
• 12+ 新增 API 端点
• SSE 流式通信实现实时进度推送
• **多引擎架构设计**，支持 Claude Code、Codex、Gemini 等

🤖 **多引擎 AI 能力矩阵**
• ✅ Claude Code - Anthropic Claude Agent SDK 深度集成
• ✅ Codex - OpenAI Codex SDK 完整支持
• 🔜 Gemini - Google Gemini 即将接入
• 🔄 对话中热切换引擎，历史记录完整保留
• 📋 任务上下文自动生成与注入
• 💬 流式输出 + Markdown 渲染
• 🗄️ 会话历史持久化 + 任务关联

📊 **数据模型优化**
• ai_sessions - AI 会话记录（支持多引擎）
• ai_messages - 消息流管理
• ai_verifications - 任务验证（v1.2.1 UI）
• buildTodoTree() 树状结构转换 O(n)

⚡ **用户体验优化**
• 任务层级可视化（主任务+子任务）
• 折叠/展开交互
• 进度条动态更新
• 快捷状态按钮（开始→暂停→完成）
• **AI 引擎选择器**（UI 一键切换）

🎯 **性能指标**
• AI 拆分响应时间：3-5秒
• 树状结构构建：O(n) 时间复杂度
• SSE 延迟：< 100ms
• 零额外依赖增加
• **多引擎并行支持**，无性能损耗

📦 **技术栈**
React 18 + TypeScript + Vite
Node.js + Express + SQLite
**Multi-Engine AI** (Claude Code + Codex) + SSE

🎨 **差异化优势**
vs 单引擎工具：**支持多引擎自由切换**
vs 纯项目管理：**AI 驱动的开发体验**
vs 通用 AI：**专注 Vibe Coding 场景**
